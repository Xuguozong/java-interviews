> 学习 JUC 时，时刻想着 JUC 是要解决什么问题（并发编程面临的资源竞争，IPC 通信协作），然后从 OS 或硬件层面去了解有哪些常用的解决方案，然后将这些解决方案对应到 JUC 中的各种类，进行分类，然后深入底层代码看看它们在 Java 中是如何处理的。

在学习所有技术时都应该要按照这个思路进行，所以计算机基础是非常重要的。
